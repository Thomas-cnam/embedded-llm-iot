# Team Organization

This document describes the initial team organization for the internship project. The division may evolve during the internship depending on progress, hardware availability, and project needs.

## Student 1

Responsibilities:

- Firmware development
- ESP32-C6 setup
- MicroPython scripts
- Sensor and actuator tests
- Hardware bring-up notes

## Student 2

Responsibilities:

- Python gateway development
- Local LLM setup in later project phases
- Logging
- Benchmarking
- Gateway-side safety validation

## Shared Responsibilities

- Documentation
- Experiments
- Final report
- GitHub repository maintenance
- Reviewing and testing project code

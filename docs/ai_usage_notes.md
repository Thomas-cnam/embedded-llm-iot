# AI Usage Notes

AI tools may be used responsibly during this project for:

- Code scaffolding
- Debugging help
- Prompt design
- Documentation editing
- Literature review support

All AI-generated code must be understood, reviewed, tested, and documented by the students before it is used in the project. AI output should not be treated as validated evidence. Hardware behavior must be confirmed on the real ESP32-C6 board and recorded in `LOG.md`.
